# Implementation Rules

## General

Prefer simple solutions.

Do not introduce complexity.

## Forbidden

- Backend
- API
- Authentication
- Cloud Sync
- Firebase
- Supabase
- Analytics
- Push Notifications

## UI

Keep screens simple.

Prioritize speed over aesthetics.

## Data

Application must work fully offline.

## Dependencies

Prefer Apple native frameworks.

Avoid unnecessary third-party libraries.
