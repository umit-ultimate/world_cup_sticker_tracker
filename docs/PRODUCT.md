# Panini Tracker

## Goal

A lightweight iOS application for managing a Panini World Cup 2026 sticker album.

The application is intended for personal use.

## Primary Problems

1. Identify missing stickers.
2. Determine whether a sticker is needed.
3. Find album pages quickly.
4. Reduce effort while checking large sticker collections.

## Non Goals

- Trading management
- Multi-user support
- Social features
- Authentication
- Cloud synchronization
- Monetization
- App Store publishing

## Core Workflow

1. User initializes album.
2. User marks missing stickers.
3. Application calculates album status.
4. User searches stickers.
5. User checks whether a sticker is needed.
